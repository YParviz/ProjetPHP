----------- Formulaires PHP -----------
Les formulaires PHP sont disponnibles dans le dossier PHP.
Pour pouvoir les tester, il vous faut une connexion à une base de donnée PostgreSQL.
Vous devez changer les identifiants de connexion dans les fichiers suivant :
- login.php
- register.php
- creation_debat.php
- signalement.php
- config.php

Vous pouvez accéder à la liste de tous les formulaires via le fichier index.html

----------- SQL -----------
Le script qui permet de créer les tables de la base de données est le fichier tables.sql
Le script qui permet de créer les tuples dans la base de données est le fichier donnees.sql
Toutes les requêtes SQL que nous avons fait se trouvent dans le fichier requetes.sql
